Programming Languages

These are some programming languages I've made

A custom object oriented programming language made with C#. Originally made for easy WInForms development with versions 1-2. Version 3 detaches from WinForms and includes awesome features like a package manager and the ability to make custom packages with C# and EZCode. It has a very unique syntax compared to other languages.

Norma is a functional based programming language built in C# made to be extremely simple. It's syntax is similar to Python and JavaScript, but has my own custom spin on it.

Calc Lang is a functional programming langauage that makes it extremely straightforward to program mathamatical expressions. It supports variables, functions, and basic mathamatical operations including modulas (%) and factorials !.
